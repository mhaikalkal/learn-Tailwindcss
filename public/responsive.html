Tailwind Uses a Mobile First Breakpoint System by Default.

Did you know that Tailwind by default is using Mobile Breakpoint System First!? You didn't right? So, make sure your website looks good in mobile, and then you can customized your website on another breakpoint system. What this means is that unprefixed utilities (like uppercase) take effect on all screen sizes, while prefixed utilities (like md:uppercase) only take effect at the specified breakpoint and above.
